# Week 9: Agents Post-Deployment

**Dates:** Mon 11/17 & Fri 11/21

---

## Objectives

Understand what happens after the agent ships — how to monitor AI-assisted systems in production, respond to incidents autonomously, and debug failures in distributed environments. Explore how multi-agent systems are transforming on-call engineering. Guests: Mayank Agarwal (CTO, Resolve) and Milind Ganjoo (Technical Staff, Resolve).

---

## Topics

- Monitoring and observability for AI systems — traces, spans, and metrics for LLM-powered services
- Automated incident response — agents that detect, triage, and resolve production issues
- Triaging and debugging — AI-assisted root cause analysis at scale
