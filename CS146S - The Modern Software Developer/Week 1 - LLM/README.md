# Week 1: Introduction to Coding LLMs and AI Development

**Dates:** Mon 9/22 & Fri 9/26

---

## Objectives

Understand what Large Language Models are at a fundamental level and learn how to interact with them effectively through prompting. This week establishes the theoretical foundation for the entire course — how LLMs are built, how they reason, and how prompt engineering shapes the quality of AI-generated code and responses.

---

## Topics

- Course logistics and overview
- What is an LLM, actually — architecture, training, and inference
- How to prompt effectively — techniques that work, pitfalls to avoid
