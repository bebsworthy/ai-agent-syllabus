# Week 4: Coding Agent Patterns

**Dates:** Mon 10/13 & Fri 10/17

---

## Objectives

Learn how to orchestrate AI coding agents effectively — when to give them full autonomy, when to stay in the loop, and what patterns lead to reliable, production-quality output. Guest: Boris Cherney, Creator of Claude Code.

---

## Topics

- Managing agent autonomy levels — from supervised to fully autonomous
- Human-agent collaboration patterns — review cycles, handoff strategies, trust calibration
