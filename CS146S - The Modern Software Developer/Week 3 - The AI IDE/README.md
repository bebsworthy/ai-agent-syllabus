# Week 3: The AI IDE

**Dates:** Mon 10/6 & Fri 10/10

---

## Objectives

Master the modern AI-powered development environment. Learn how to manage context effectively, write PRDs that guide agents precisely, and configure IDE integrations for maximum productivity. Guest: Silas Alberti, Head of Research at Cognition (maker of Devin).

---

## Topics

- Context management and code understanding — what agents see and why it matters
- PRDs for agents — specs as executable intent
- IDE integrations and extensions — toolchains for the AI-first developer
