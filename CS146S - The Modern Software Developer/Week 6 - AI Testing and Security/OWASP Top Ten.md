# OWASP Top Ten Web Application Security Risks

**Source:** https://owasp.org/www-project-top-ten/

## Overview

The OWASP Top 10 serves as a foundational security awareness document for developers and organizations. It represents a broad consensus about the most critical security risks to web applications.

The most current version is **OWASP Top Ten 2025**. Previous releases include the 2021 and 2017 editions.

## Key Purpose

Organizations should adopt this document to minimize security vulnerabilities in their web applications. The Top 10 is recognized globally as a crucial first step toward developing more secure software and shifting organizational culture toward secure coding practices.

## Translation Efforts

### 2025 Status
Translations are currently in progress for the 2025 version.

### 2021 Completed Translations
Nine languages have been fully translated, including Arabic, Spanish, French, Indonesian, Italian, Japanese, Portuguese (Brazil), and two Chinese variants.

### Historical Translations
Earlier versions (2017, 2013, and 2010) have been translated into numerous additional languages including German, Korean, Russian, Hebrew, and Vietnamese.

## 2025 Data Analysis Initiative

The project is collecting comprehensive vulnerability data from multiple sources to inform the 2025 rankings.

**Contribution Period:** Through July 31, 2025, accepting data from 2021-2024

**Submission Methods:**
- Email CSV/Excel files to designated address
- Upload via provided link

**Key Requirements:**
- Testing timeframe
- Number of applications tested
- CWE (Common Weakness Enumeration) lists with occurrence counts

Contributors can submit known, pseudo-anonymous, or fully anonymous data, with verification status clearly noted.
