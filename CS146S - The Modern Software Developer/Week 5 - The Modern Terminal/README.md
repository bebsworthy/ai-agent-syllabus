# Week 5: The Modern Terminal

**Dates:** Mon 10/20 & Fri 10/24

---

## Objectives

Explore how AI is transforming the command line — from intelligent autocomplete to fully agentic terminal workflows. Learn how tools like Warp are reinventing the terminal for the AI era and how to script and automate development tasks with AI assistance. Guest: Zach Lloyd, CEO of Warp.

---

## Topics

- AI-enhanced command line interfaces — what the next-generation terminal looks like
- Terminal automation and scripting — combining AI with shell power
