# Week 2: The Anatomy of Coding Agents

**Dates:** Mon 9/29 & Fri 10/3

---

## Objectives

Understand how coding agents are architected from the ground up — what makes them work, how they use tools and call functions, and how the Model Context Protocol (MCP) enables agents to connect to external systems. Students will build a coding agent from scratch and implement a custom MCP server.

---

## Topics

- Agent architecture and components — planning loops, memory, tool dispatch
- Tool use and function calling — structured interfaces for LLM action
- MCP (Model Context Protocol) — the emerging standard for agent-tool connectivity
