# Week 7: Modern Software Support

**Dates:** Mon 11/3 & Fri 11/7

---

## Objectives

Learn how AI is transforming code review, debugging, and documentation. Understand what makes AI code trustworthy, how to review AI-generated code effectively, and how intelligent documentation tools raise the quality bar across engineering teams. Guest: Tomas Reimers, CPO of Graphite.

---

## Topics

- What AI code systems can we trust — evaluating confidence and reliability in generated code
- Debugging and diagnostics — AI-assisted root cause analysis
- Intelligent documentation generation — from comments to full docs, automated
