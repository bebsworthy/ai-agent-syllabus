## Who this is for

Developers who want to be productive with Claude Code today. You have a real codebase, real deadlines, and no time for theory. Each module produces something you will use immediately.

## How it works

8 modules. Each one:
- Tells you what to open
- Walks you through an exercise on **your own codebase**
- Produces a tangible artifact (a file, a skill, a connection, a workflow)
- Links to the full playbook if you want to understand the "why" later

**Total: ~4 hours 20 minutes**

## Prerequisites

- A real project codebase (not a tutorial project)
- Admin access to install software
- A Claude account (free tier works; paid unlocks larger context windows)
- Git installed
