## How each module works

Every module follows the same pattern:

1. **Pre-work** (15–20 min) — Self-paced theory reading that builds a mental model
2. **Workshop** (45–60 min) — Self-directed hands-on session applying the theory
3. **Takeaway** — A concrete artifact, habit, or workflow you use from that day forward
